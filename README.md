# Rising Waters

## Flood Prediction Using Machine Learning

### Team Details
- Project Name: Rising Waters
- Domain: Artificial Intelligence & Machine Learning
- Platform: SmartBridge

### Project Description
Rising Waters is a machine learning-based flood prediction system that predicts flood risk using environmental and weather-related parameters.

### Technologies Used
- Python
- Flask
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- IBM Cloud
